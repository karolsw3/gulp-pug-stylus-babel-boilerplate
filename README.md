<b>Gulp 4</b> ready boilerplate with pug, stylus and babel! Includes auto page reload.

## Installation

1. Install dependencies  
	```
	npm install
	```
2. Run gulp
	```
	gulp
	```
3. Open http://localhost:8080

Every time you save the code the page automatically reloads.
Happy coding!
